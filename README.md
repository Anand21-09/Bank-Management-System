# Bank Management System (ATM Simulator)

## About

Bank Management System is a Java desktop application that simulates the basic functionality of an ATM. Users can create accounts, log in securely, deposit and withdraw money, check account balances, view mini statements, and manage their PIN through a simple graphical interface.

## Tech Stack

* Java
* Java Swing
* AWT
* MySQL
* JDBC
* Object-Oriented Programming (OOP)
