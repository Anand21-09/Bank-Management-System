# 🏦 Bank Management System (ATM Simulator)

A desktop-based Banking Management System developed in Java that simulates the functionality of an ATM. The application provides users with secure banking operations such as account creation, login authentication, cash withdrawal, deposits, balance inquiry, transaction history, PIN changes, and mini statements.

## 📌 Features

### 👤 User Authentication

* Secure Login System
* New Account Registration
* Multi-Step Signup Process
* PIN Verification

### 💰 Banking Operations

* Cash Deposit
* Cash Withdrawal
* Fast Cash Withdrawal
* Balance Inquiry
* Transaction History
* Mini Statement Generation
* PIN Change Facility

### 🏦 Account Management

* Create New Bank Account
* Store Customer Information
* Manage Account Credentials

## 🚀 Functionalities

### Account Registration

Users can create a new bank account through a multi-step registration process.

### Login Authentication

Users can securely log in using their card number and PIN.

### Deposit Money

Allows users to deposit money into their account.

### Withdraw Money

Users can withdraw available funds from their account.

### Fast Cash

Provides quick withdrawal options for predefined amounts.

### Balance Enquiry

Displays the current account balance.

### Mini Statement

Shows recent transactions performed by the user.

### PIN Change

Allows users to update their ATM PIN securely.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/Bank-Management-System.git
```

### Navigate to Project

```bash
cd Bank-Management-System
```

### Configure Database

1. Create a MySQL database.
2. Import the required tables.
3. Update database credentials in:

```java
Conn.java
```

### Run Application

Compile and execute:

```bash
javac *.java
java Login
```
